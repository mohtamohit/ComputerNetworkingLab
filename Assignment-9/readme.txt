There are two folders client and server.

Steps to execute :
1. compile and run server.c
2. compile and run client.c
3. compile and run client2.c
3. Then, type your message in client.c 's process.
4. You will see that the message will appear in client2.c 's process.
5. Repear 3 and 4 or vice versa to continue chatting.
